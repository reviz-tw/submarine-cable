# 台灣海纜討論第一版來源名單

## 目的

這份名單是給 `Facebook / Threads` 抓取第一輪使用的入口清單。

目前關鍵字固定只用：

- `海底電纜`
- `海纜`

策略上分兩層：

1. `Threads`
   - 先用這兩個詞搜尋
   - 再從命中結果抽出值得補深的帳號
2. `Facebook`
   - 不靠全站搜尋當主軸
   - 直接從這份指定來源名單開抓

## 使用原則

- `P1`：第一批就要抓，最可能直接出現海纜相關討論
- `P2`：第二批補抓，作為延伸來源
- `P3`：第三批補抓，偏脈絡型或事件型來源

## Facebook 第一批來源

### P1: 官方與基礎設施來源

這批來源的價值在於：

- 最容易出現澄清、修復進度、政策說明
- 發文量相對可控
- 適合當事件時間軸主幹

#### 1. 數位發展部 moda

- 類型：政府機關
- 優先度：`P1`
- 理由：
  - 海纜事件若涉及對外通訊韌性、斷纜澄清、政策回應，數發部最常直接表態
  - 已可找到海纜相關公開貼文
- 建議抓法：
  - 先回看 `2023-01` 起所有含海纜討論的貼文
  - 再對海纜事件貼文補抓留言

#### 2. 中華電信相關公開頁面

- 類型：電信業者
- 優先度：`P1`
- 理由：
  - 台馬海纜、通訊備援、修復說明常會跟中華電信脈絡連動
  - 即使不是每次都直接發海纜文，也值得作為固定回訪來源
- 注意：
  - 中華電信在 Facebook 上有多個子頁面，需先確認哪個是最常發布公共議題或公司訊息的主要頁
- 建議抓法：
  - 先確認主頁與新聞型貼文來源
  - 再對 `海底電纜` / `海纜` 命中的公司貼文補留言

#### 3. 連江縣政府 / 馬祖地方政府相關頁面

- 類型：地方政府
- 優先度：`P1`
- 理由：
  - 台馬海纜事件直接牽涉馬祖居民用網與地方民生
  - 地方政府頁面可能比中央更早出現生活影響、應變措施、居民反應
- 建議抓法：
  - 先找縣政府官方頁
  - 再找縣長或官方新聞發布頁

#### 4. 馬祖地方新聞 / 在地資訊頁

- 類型：地方媒體 / 地方資訊
- 優先度：`P1`
- 理由：
  - 馬祖是台灣海纜事件最具體的地方場景
  - 地方頁面更容易出現居民日常語言下的討論與抱怨
- 優先關注：
  - 馬祖日報
  - 馬祖在地資訊頁
  - 馬祖社團或地區型公共討論頁

### P1: 台灣主要媒體來源

這批來源的價值在於：

- 重大海纜事件發生時，最容易形成高互動貼文
- 留言區可觀察輿論怎麼詮釋事件

#### 5. 中央社 CNA

- 類型：主流媒體
- 優先度：`P1`
- 理由：
  - 海纜事件常有整理型報導
  - 官方資訊引用率高
  - 留言區較容易看到一般民眾與政治詮釋混合

#### 6. 報導者 The Reporter

- 類型：深度媒體
- 優先度：`P1`
- 理由：
  - 已可確認有海纜專題貼文
  - 深度報導的留言區通常有較完整的問題意識與補充資訊

#### 7. 公視新聞網 PNN

- 類型：公共媒體
- 優先度：`P1`
- 理由：
  - 災害、基礎建設、公共政策與地方議題 coverage 穩定

#### 8. ETtoday 新聞雲

- 類型：主流媒體
- 優先度：`P1`
- 理由：
  - 即時事件多
  - 互動量通常高
  - 留言區比較容易看到快速反應與爭議性說法

#### 9. TVBS 新聞

- 類型：主流媒體
- 優先度：`P1`
- 理由：
  - 海纜事件若進入主流政治或國安敘事，TVBS 常會有討論熱度

#### 10. 三立新聞

- 類型：主流媒體
- 優先度：`P1`
- 理由：
  - 台灣公共議題互動量高
  - 可觀察不同政治受眾的留言語氣

#### 11. 聯合新聞網 / 聯合報系

- 類型：主流媒體
- 優先度：`P1`
- 理由：
  - 經常涵蓋基礎設施、國安、地方與中央政策

#### 12. 自由時報

- 類型：主流媒體
- 優先度：`P1`
- 理由：
  - 海纜事件若與中國因素、國安論述連結，常有高互動貼文

## Facebook 第二批來源

### P2: 科技 / 國安 / 地緣政治導向來源

這批來源的價值在於：

- 不一定最早報導事件
- 但很容易出現分析型或解讀型討論

#### 13. 科技媒體

- 類型：科技 / 產業媒體
- 優先度：`P2`
- 例子：
  - INSIDE
  - iThome
  - TechOrange
- 理由：
  - 若海纜議題轉向通訊韌性、基礎設施、數位政策，這些頁面可能發文

#### 14. 軍事 / 國安 / 地緣政治評論頁

- 類型：公共評論 / 議題頁
- 優先度：`P2`
- 理由：
  - 海纜議題在台灣常被放進灰色地帶衝突與關鍵基礎設施脈絡
  - 留言區容易出現強烈敘事與推論
- 注意：
  - 這類來源雜訊也較高，適合第二批而非第一批

#### 15. 台灣政策觀察 / 公共議題頁

- 類型：公共議題頁
- 優先度：`P2`
- 理由：
  - 可補足政府、媒體與地方生活之間的中介討論層

## Facebook 第三批來源

### P3: 地方社群與事件型來源

這批來源的價值在於：

- 可以抓到更生活化、更地方性的回應
- 但頁面較分散、噪音也較高

#### 16. 馬祖 / 離島社團

- 類型：公開社團
- 優先度：`P3`
- 理由：
  - 若事件影響居民生活，社團裡最容易看到實際抱怨與使用經驗
- 注意：
  - 只抓公開社團
  - 先確認是否可穩定回捲與讀取內容

#### 17. 地方社區 / 爆料型公開頁

- 類型：地方公共討論頁
- 優先度：`P3`
- 理由：
  - 容易出現第一手生活反應
  - 但噪音高，需要人工挑文

## Threads 抓取入口

## 第一層：直接搜關鍵字

固定只搜：

- `海底電纜`
- `海纜`

要觀察的不是只有貼文本身，還包括：

- 哪些帳號反覆出現
- 哪些貼文集中在某些事件日期
- 哪些帳號明顯來自台灣媒體 / 政府 / 在地社群

## 第二層：從搜尋結果抽帳號

從 Threads 搜尋命中結果裡，優先保留以下帳號類型：

### A. 台灣媒體帳號

- CNA
- 報導者
- 公視 / PNN
- ETtoday
- 其他主流媒體

### B. 台灣政府 / 公共機構帳號

- 數位發展部
- 地方政府
- 與通訊基礎設施相關的公共單位

### C. 台灣在地議題帳號

- 馬祖或離島公共討論帳號
- 地方觀察者
- 地方記者

### D. 台灣公共評論帳號

- 長期談國安、數位韌性、台海風險的人
- 只要多次命中 `海底電纜` / `海纜`，就值得補抓歷史

## 實際抓取順序

### Round 1

- Threads 搜尋 `海底電纜`
- Threads 搜尋 `海纜`
- 整理命中貼文 URL、作者、時間
- 抽出高頻台灣帳號

### Round 2

- 抓 Facebook `P1` 來源
- 每個來源回捲看是否有海纜相關貼文
- 命中貼文先抓主文與基本互動數

### Round 3

- 對 Threads 命中的高頻台灣帳號補抓歷史
- 對 Facebook 命中的重要貼文補抓留言

### Round 4

- 再補 Facebook `P2`、`P3` 來源
- 對高價值舊文做留言回訪

## 建議的最小可行名單

若只想先抓一小批，先從這 8 類來源開始：

1. 數位發展部 moda
2. 中華電信主頁
3. 連江縣政府 / 馬祖地方政府頁
4. 馬祖地方新聞頁
5. 中央社 CNA
6. 報導者 The Reporter
7. 公視新聞網 PNN
8. ETtoday 新聞雲

## 備註

- 這份名單的重點是「第一版能開工」，不是一次列出所有可能頁面。
- 真正開始抓取後，應該把命中頻率高的來源再升級為固定回訪來源。
- 若某個來源幾乎不發海纜文，就降級或移出，避免浪費抓取時間。
